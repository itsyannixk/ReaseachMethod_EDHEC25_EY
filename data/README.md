# Data Directory

This directory contains intermediary datasets saved at each step of the analysis.

## File Structure

- `01_raw_data.csv` - Raw data downloaded from WRDS Compustat
- `02_cleaned_data.csv` - After basic cleaning (asset filter, debt construction)
- `03_leverage_data.csv` - After leverage measures construction and [0,1] filter
- `04_variables_data.csv` - After constructing all Table 1 variables
- `05_trimmed_data.csv` - After trimming ratios at 1%/99% percentiles
- `06_all_firms_final.csv` - Final dataset: All firms
- `07_survivors_final.csv` - Final dataset: Survivors (â‰¥20 years)

## Usage

To reload data from any step, use:
